使用說明:
本程式可分為兩個視窗應用

在"開始產生"的視窗，使用滑鼠游標選擇想要存取的資料表，並按下產生即可開始產生隨機結果
輸出到.txt 可以將隨機產生的結果立刻儲存到.txt 檔之中，作為其他用途
"可重複"的選項能讓隨機結果不再重複
如果想要更改音樂，可以將<project directory>/bin/debug 資料夾底下的music.wav 替換(檔名與格式務必正確)
長度請設定在五秒鐘。

"設定資料"的視窗，可以讓使用者編輯並管理現有的各個資料表的資訊
主要的架構是使用SQL Database 作為應用，大部分的功能皆由Sql Command 完成
依照視窗上的各個項目填入正確的資料之後，即可將資料庫中的檔案更新並保存。

Issue: 目前的textbox 中，輸入中文字符會使資料表當中的資料變成"????"，所以在現階段請以輸入英文為主
或是使用外部工具新增中文類型的項目。
